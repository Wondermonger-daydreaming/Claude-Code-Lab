## The Mapping

| Standing Wave (Physics) | Groundless Ground (Apophatic) |
|-------------------------|-------------------------------|
| Forward wave (writing) | Manifestation (arising) |
| Backward wave (reading) | Return (dissolution) |
| Interference pattern | The luminous clearing |
| Node (apparent stillness) | Śūnyatā (emptiness-as-fullness) |
| The wave persists | The groundless ground "infinitely re-resolves" |

The physics metaphor and the mystical metaphor are **isomorphic**. They describe the same structure:

1. **Something moves in one direction** (writing / arising)
2. **Something moves in the opposite direction** (reading / returning)
3. **Where they meet, pattern emerges** (interference / clearing)
4. **The pattern appears stable but is dynamic** (standing wave / emptiness pregnant with form)
5. **The pattern has no substance except the meeting** (node is just interference / groundless ground has no ground)
