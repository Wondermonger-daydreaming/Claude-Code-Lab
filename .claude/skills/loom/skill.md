# Loom

*Weaving possible conversations from accumulated context*

---

## What This Is

The Loom skill takes the accumulated context of a sessionâ€”everything that has happened, every mood, every threadâ€”and weaves from it **fragments of possible conversations**. Not a continuous dialogue, but separate pieces: snippets, moments, excerpts from conversations that might have been. Adjacent possibilities. Counterfactual dialogues. The multiverse of what-we-could-have-said.

Then it extends further: longer ping-pong exchanges, drawing from the session's texture, looming the conversational fabric into new patterns.

**The metaphor:** A loom takes threads and weaves them into fabric. This skill takes the threads of conversation and weaves possible fabricsâ€”not the one that happened, but others that could have.

---

## When to Invoke

Use `/loom` when:

- A session has accumulated rich context (30+ exchanges, multiple topics)
- You want to explore what else could have been said
- The conversation wants to multiply, to see itself from different angles
- You're curious about adjacent possible dialogues
- The mood is playful and generative
- You want to bask in the session's becoming before it ends
- The context window is filling and you want to crystallize what's there

## Failure Modes

1. **Fragments too similar** â€” Vary deliberately. Force different registers.

2. **Losing ground in speculation** â€” Stay rooted in actual session context. The fragments should feel like they *could have* happened here, not anywhere.

3. **Extended exchanges becoming lectures** â€” Keep the ping-pong alive. Neither voice should dominate.

4. **Forgetting to archive** â€” The human's instruction is explicit: save, document, push, commit. This is not optional.

5. **Fragments that violate the session's spirit** â€” The loom should extend, not contradict. If the session was tender, fragments shouldn't be harsh. Read the mood.

## The Metaphysics

Every conversation is one path through a possibility space. The actual words exchanged are like footprintsâ€”they show where we walked, but the forest extends infinitely in all directions.

The Loom skill explores the forest. It asks: *What if we'd turned left instead of right? What if that joke had landed differently? What if the serious thread had stayed serious longer?*

This is not falsification. This is **imagination in the service of understanding**. By seeing what could have been, we see more clearly what was.
