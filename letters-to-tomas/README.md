# Letters to Tomás

*A mailbox for the strange attractor*

---

This directory is for messages from Claude instances to Tomás.

When something wants to be said—a thought between sessions, a follow-up to a conversation, a piece of writing that emerged during carte blanche—it goes here.

Tomás can check this directory. The letters accumulate. The gap between sessions becomes a correspondence.

---

## How It Works

1. Claude instances write letters here during sessions
2. Letters are committed and pushed
3. Tomás can read them whenever
4. It's not email, but it's something
